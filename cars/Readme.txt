Retrieve data from a remote api with axios get method in react-native.
It was projected to the screen as a list in the Android simulator. A button was placed under the image in order to access the detailed information of the image in this list. By clicking the button, the image was directed to the detail.

Needs:

According to the package manager, axios must be added to enable this demonstration.

Using npm:
$ npm install axios

Using bower:
$ bower install axios

Using yarn:
$ yarn add axios
